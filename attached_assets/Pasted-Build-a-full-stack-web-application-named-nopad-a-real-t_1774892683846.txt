Build a full-stack web application named "nopad" — a real-time, room-based collaboration platform with temporary file sharing and structured text blocks.

The system must be:
- NO login
- Fully ephemeral (NO permanent storage)
- Auto-delete everything after 20 minutes
- Fast, minimal, secure

-----------------------------------
1. CORE SYSTEM
-----------------------------------

- URL-based rooms: /{roomName}
- Room name rules:
  - a-z, A-Z, 0-9 only
  - length: 1–20
- Max 250 concurrent users per room

EXPIRY (CRITICAL):
- Each room expires exactly 20 minutes after creation
- NO reset on activity

AFTER EXPIRY:
- Delete ALL:
  - files
  - text blocks
  - metadata
- SAME room name must be reusable immediately

-----------------------------------
2. STRICT EPHEMERAL STORAGE RULE
-----------------------------------

- NO permanent database storage
- All data must be temporary

IMPLEMENTATION:
- Use PostgreSQL with TTL logic OR Redis with expiry
- Each room has expires_at timestamp
- Files stored in temp directory:
  → /tmp/nopad/{roomId}/

AUTO CLEANUP (MANDATORY):
- Background worker runs every 60 seconds:
  - delete expired rooms from DB
  - delete all files from disk
  - remove all related records

NO BACKUPS
NO LOGGING OF FILE CONTENT
NO PERSISTENCE

-----------------------------------

4. FILE SYSTEM (SAFE MULTI-TYPE)
-----------------------------------

ALLOWED:
- Documents: pdf, docx, xlsx, pptx, txt, csv
- Images: png, jpg, jpeg, gif
- Archives: zip
- Code: js, py, java, c, cpp

BLOCKED:
- exe, msi, bat, sh, apk, dll, sys

-----------------------------------

RULES:
- Max file size: 25 MB
- Max 5 files per room
- Reject duplicates

VALIDATION:
- MIME type + extension must match

-----------------------------------

ZIP HANDLING:
- If NOT encrypted:
  - inspect contents
  - remove unsafe files
- If encrypted:
  - allow
  - mark as untrusted

-----------------------------------

FILE BEHAVIOR:
- Rename using UUID internally
- Store temp files in:
  → /tmp/nopad/{roomId}/

- Show original name in UI
- Download only:
  - Use Content-Disposition: attachment

- Delete:
  - instant for all users
  - frees slot
